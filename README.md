## A Chat App is basically the first step toward building a WhatsApp clone.

Think of it like this:

     Chat App

        ↓
    WhatsApp Clone

        ↓
    Discord Clone


The chat app contains only the core messaging functionality.


## Features :

### User Authentication -

      Login
      Signup
      JWT Authentication
      
### Chat -

     Send message
     Receive message instantly
     Chat history

### Online Users -

    🟢 Abdul
    ⚫ Aman
    🟢 Rahul
    
### Typing Indicator

    Abdul is typing...
    
### Timestamps

    Hello!
    10:35 PM
    
### Scroll to Latest Message

    Exactly like WhatsApp.
